Claude Code Hub (CCH) 是一个团队级多供应商 AI Coding 代理调度平台，可作为服务器部署的多租户工具，让团队通过单一接口管理多个 AI 供应商。

## 核心功能

- **负载均衡** — 支持权重、优先级和分组调度
- **多供应商支持** — Claude、Codex、Gemini 和 OpenAI 兼容服务
- **速率限制** — RPM 限制、金额限制和并发会话控制
- **实时监控** — 使用指标和供应商健康状态仪表板
- **会话管理** — 5 分钟上下文缓存和粘性路由
- **OpenAI 兼容** — 通过 `/v1/chat/completions` 端点访问

## 适用场景

- 敏捷开发团队
- AI 密集型开发环境
- 预算有限的初创公司
- 需要合规审计的企业
