Claude Code 是 Anthropic 官方出品的 AI 编程 CLI 工具，可在终端中使用 Claude 进行代码编写和调试。

## 安装方式

### Windows 平台

使用 PowerShell 安装：

```powershell
npm install -g @anthropic-ai/claude-code
```

### macOS / Linux 平台

使用终端安装：

```bash
npm install -g @anthropic-ai/claude-code
```

或使用 Homebrew：

```bash
brew install claude-code
```

## 首次启动

<Callout type="important">
  使用第三方中转服务时，Claude Code 首次启动可能会报错："Unable to connect to Anthropic services"。这是正常现象，需要跳过初始确认。
</Callout>

### 方法一：使用 CC-Switch 工具（推荐）

CC-Switch 是一个图形化配置工具，可以一键完成所有配置。

1. 下载并安装 [CC-Switch](https://github.com/farion1231/cc-switch)
2. 添加 Nebula API 配置
3. 切换到 Nebula API 服务商
4. 启动 Claude Code

### 方法二：手动配置

编辑 `~/.claude.json` 文件，添加以下内容：

```json
{
  "hasCompletedOnboarding": true
}
```

然后配置环境变量：

```bash
# macOS / Linux
export ANTHROPIC_AUTH_TOKEN="your-api-key"
export ANTHROPIC_BASE_URL="https://api.nebula.ai"

# Windows PowerShell
$env:ANTHROPIC_AUTH_TOKEN = "your-api-key"
$env:ANTHROPIC_BASE_URL = "https://api.nebula.ai"
```

## 验证安装

启动 Claude Code：

```bash
claude
```

如果看到 Claude Code 界面，说明安装成功。

## 配置说明

### 环境变量

| 变量名 | 说明 |
|--------|------|
| `ANTHROPIC_AUTH_TOKEN` | 您的 API Key |
| `ANTHROPIC_BASE_URL` | API 端点地址 |

### 配置文件

配置文件位于 `~/.claude.json`，主要参数：

```json
{
  "hasCompletedOnboarding": true,
  "apiKey": "your-api-key",
  "baseUrl": "https://api.nebula.ai"
}
```

<Callout type="tip">
  推荐使用环境变量方式配置，更安全且便于管理。
</Callout>

## 常见问题

### 连接失败？

1. 检查网络连接是否正常
2. 确认 API Key 是否正确
3. 确认账户余额是否充足
4. 检查是否选择了正确的令牌组

### 响应慢？

可以尝试：
- 使用 CC-Switch 的速度测试功能选择最快的节点
- 检查网络状况

### 环境变量不生效？

- **Windows**: 重启 PowerShell 或命令提示符
- **macOS/Linux**: 运行 `source ~/.zshrc` 或 `source ~/.bashrc`
