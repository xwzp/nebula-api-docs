## 充值方式

Nebula API 平台支持以下在线充值方式：

- **支付宝**
- **微信支付**

### 充值步骤

1. 登录 Nebula API 控制台
2. 进入 **"充值"** 页面
3. 选择充值金额（10元、50元、100元等预设金额，或自定义金额）
4. 选择支付方式
5. 完成支付

<Callout type="info">
  两种支付方式均为即时到账。如超过 5 分钟未到账，请发送邮件联系。
</Callout>

## 余额管理

在控制台可以查看：

- 当前余额
- 充值记录
- 消费明细
- 账单详情

## 定价说明

我们采用按量付费模式，根据实际使用的 token 数量计费。

<Callout type="tip">
  价格低于官方定价，通过运营优化让更多开发者用得起优质 AI 工具。
</Callout>

## 发票服务

- 实际消费达 **500 元** 后可申请发票
- 支持普票和专票
- 大额开票请发送邮件联系

### 申请流程

1. 确认消费金额达到开票门槛
2. 在控制台提交开票申请
3. 填写发票信息
4. 等待处理（通常 1-3 个工作日）

## 退款规则

- 未使用的余额可申请退款
- 已消费部分不支持退款
- 退款需扣除一定手续费
- 退款周期约 3-5 个工作日

<Callout type="important">
  退款前请确认账户内无待处理的消费订单。
</Callout>

## 常见问题

### 充值失败怎么办？

1. 检查网络连接是否正常
2. 确认支付方式可用
3. 尝试更换支付方式
4. 发送邮件获取帮助

### 余额不足会怎样？

余额不足时，API 请求会返回错误。请及时充值以继续使用服务。

### 企业/大额充值

企业用户或大额充值需求，请发送邮件至 support.ai@nebulatrip.com：

- 可获得更优惠的价格
- 提供发票支持
