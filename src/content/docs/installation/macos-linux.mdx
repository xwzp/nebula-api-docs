<Callout type="important">
  Claude Code、CodeX、Gemini CLI 都需要 Node.js 18+ 运行环境。
</Callout>

## 安装方法

### 方法一：使用 Homebrew（推荐）

Homebrew 是 macOS 最流行的包管理器。

```bash
brew install node
```

安装完成后验证：

```bash
node --version
npm --version
```

<Callout type="tip">
  如果 Homebrew 下载慢，可以切换到国内镜像源。
</Callout>

### 方法二：官方安装包

1. 访问 [Node.js 官网](https://nodejs.org/)
2. 下载 macOS LTS 版本（.pkg 文件）
3. 双击运行安装程序
4. 按提示完成安装

### 方法三：使用 nvm

nvm (Node Version Manager) 可以管理多个 Node.js 版本：

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

重启终端后安装 Node.js：

```bash
nvm install 18
nvm use 18
nvm alias default 18
```

## 验证安装

```bash
node --version   # 应输出 v18.x.x
npm --version    # 应输出 9.x.x
```

## 常见问题

### Homebrew 下载慢？

可以切换到国内镜像源：

```bash
# 替换 Homebrew 镜像
export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.ustc.edu.cn/brew.git"
brew update
```

### 权限问题？

如果遇到 npm 全局安装权限问题，推荐使用 nvm 代替 Homebrew 安装，nvm 自动处理权限问题。

或者使用以下命令修复：

```bash
sudo chown -R $(whoami) ~/.npm
```

<Callout type="important">
  不建议使用 `sudo npm install`，这可能导致后续权限问题。
</Callout>

## 配置环境变量

设置 API Key 环境变量：

```bash
# 添加到 ~/.zshrc 或 ~/.bashrc
export ANTHROPIC_AUTH_TOKEN="your-api-key-here"
export ANTHROPIC_BASE_URL="https://ai.nebulatrip.com"
```

然后重新加载配置：

```bash
source ~/.zshrc   # 或 source ~/.bashrc
```
