<Callout type="important">
  Claude Code、CodeX、Gemini CLI 都需要 Node.js 18+ 运行环境。请确保安装 LTS（长期支持）版本以获得最佳稳定性。
</Callout>

## 环境要求

- Windows 10 或更高版本（64 位）
- 管理员权限
- 至少 500MB 可用磁盘空间
- 网络连接

## 安装方法

### 方法一：官方安装包（推荐新手）

最简单的方式，下载安装包按向导操作即可。

1. 访问 [Node.js 官网下载页面](https://nodejs.org/en/download/)
2. 下载 **Windows Installer (.msi)** LTS 版本
3. 运行安装程序，按默认选项安装
4. 完成安装

<Callout type="tip">
  安装过程中建议勾选 **"Automatically install the necessary tools"**，自动配置构建工具。
</Callout>

### 方法二：包管理器

适合熟悉命令行的用户，可以选择以下任一方式：

**使用 Winget：**

```powershell
winget install OpenJS.NodeJS.LTS
```

**使用 Chocolatey：**

```powershell
choco install nodejs-lts
```

**使用 Scoop：**

```powershell
scoop install nodejs-lts
```

## 验证安装

打开 PowerShell 或命令提示符，执行以下命令：

```bash
node --version
npm --version
```

预期输出：

```
v18.18.0
9.8.1
```

<Callout type="info">
  版本号可能略有不同，只要 Node.js 是 18.x 或更高版本即可。
</Callout>

## 常见问题

### 命令未找到？

如果提示 `node` 命令未找到：

1. 重启终端或命令提示符
2. 检查环境变量 PATH 是否包含 Node.js 安装路径（通常是 `C:\Program Files\nodejs\`）

### 安装失败？

尝试以下方法：

1. 右键点击安装包，选择 **"以管理员身份运行"**
2. 暂时关闭杀毒软件
3. 确保网络连接正常

### 使用 nvm-windows 管理多版本

如果需要管理多个 Node.js 版本，推荐使用 [nvm-windows](https://github.com/coreybutler/nvm-windows)：

```powershell
nvm install 18
nvm use 18
```
