## 账户相关

### 如何注册账号？

访问官网 [https://ai.nebulatrip.com/](https://ai.nebulatrip.com/)，点击注册按钮，使用邮箱或第三方账号完成注册。

### 忘记密码怎么办？

在登录页面点击"忘记密码"，通过注册邮箱重置密码。

### 如何开发票？

实际消费达 500 元后可申请发票（普票或专票均可），大额开票请发送邮件联系。

## 访问问题

### 页面白屏/无法访问？

通常是浏览器广告拦截插件导致的，请将 `ai.nebulatrip.com` 添加到插件白名单中。

常见拦截插件：
- AdBlock
- uBlock Origin
- AdGuard

### 连接超时？

1. 检查网络连接
2. 尝试刷新页面
3. 清除浏览器缓存
4. 尝试更换浏览器

## 环境配置

### Node.js 版本要求？

<Callout type="important">
  需要 Node.js 18+ 版本。可使用 `node -v` 命令查看当前版本。
</Callout>

### npm 安装慢？

可以切换到国内镜像源：

```bash
npm config set registry https://registry.npmmirror.com
```

### 环境变量不生效？

- **Windows**: 需要重启 PowerShell 或命令提示符
- **macOS/Linux**: 运行 `source ~/.zshrc` 或 `source ~/.bashrc`

## 工具使用

### 不同工具需要不同的令牌吗？

<Callout type="info">
  是的。Claude Code、CodeX、Gemini CLI 需要各自独立的令牌，请在创建 Key 时选择对应的令牌组。
</Callout>

### Claude Code 启动失败？

1. 确认已完成初始化配置（`hasCompletedOnboarding: true`）
2. 检查环境变量是否正确设置
3. 确认账户余额充足
4. 尝试使用 CC-Switch 工具配置

### API 调用返回错误？

常见错误及解决方案：

| 错误码 | 说明 | 解决方案 |
|--------|------|----------|
| 401 | 认证失败 | 检查 API Key 是否正确 |
| 403 | 无权限 | 检查令牌组是否匹配 |
| 429 | 请求过多 | 降低请求频率 |
| 500 | 服务器错误 | 稍后重试 |
