Alma 是一款功能强大且界面精美的 AI 客户端工具，集成了代码编写、终端操作、Git 管理、浏览器等多种功能。

## 功能特点

- **代码编写** — 智能代码生成和补全
- **终端操作** — 集成终端，直接执行命令
- **Git 管理** — 可视化 Git 操作
- **浏览器集成** — 内置浏览器支持
- **多模型支持** — 支持 Claude、Gemini 等多种模型
- **自定义供应商** — 可接入自定义 API 提供商

## 安装与配置

### 第一步：安装 Alma

访问 [Alma 官网](https://alma.now/) 下载并安装适合你操作系统的版本。

### 第二步：设置中文界面

1. 打开 Alma 应用
2. 进入 **设置 → 通用 → 语言**
3. 选择「中文」

### 第三步：添加自定义供应商

1. 点击 **选择供应商 → Add Custom Provider**
2. 填写配置信息：
   - **Provider Name**: 自定义名称
   - **Base URL**: `https://api.nebula.ai/v1`
   - **API Key**: 您的 API Key
3. 点击 **Fetch** 获取可用模型

<Callout type="tip">
  - Claude 模型：使用逆向分组的 API Key
  - Gemini 模型：使用 gemini 分组的 API Key
</Callout>

## 新建聊天与使用

### 重要设置

<Callout type="important">
  上下文范围设置非常重要！**强烈建议选择「全选」而非「自动」！**

  选择「自动」时模型可能会产生幻觉问题，引用不存在的文件或生成错误的代码。
</Callout>

### 使用技巧

**代码编写**
- 描述你的需求，Alma 会生成对应代码
- 支持多文件操作和项目级别的修改

**终端操作**
- 集成终端直接执行命令
- 支持查看命令执行结果

**Git 管理**
- 可视化查看 Git 状态
- AI 辅助编写提交信息

## 最佳实践

### 选择合适的模型

- **Claude Sonnet** — 适合日常编码任务，速度快
- **Claude Opus** — 适合复杂的架构设计和难题
- **Gemini** — 适合需要多模态能力的任务

### 合理使用上下文

- 对于小型项目，可以「全选」所有文件
- 对于大型项目，选择相关的模块和文件
- 定期清理不相关的上下文

## Alma 和 Claude Code 对比

| 特性 | Alma | Claude Code |

| 适用场景 | 全功能开发环境 | 终端快速操作 |
| 集成功能 | 终端、Git、浏览器等 | 纯命令行工具 |
