欢迎来到 Nebula API！我们致力于为开发者提供稳定、高效的 AI 编程工具中转服务。

## 什么是中转站？

中转站即您和目标服务商之间的一座桥梁。通过 Nebula API，您可以：

- **免翻墙直连** — 无需任何科学上网工具，稳定可靠
- **价格实惠** — 低于官方定价，让更多开发者用得起
- **专业运维** — 专人维护、一对一售后、问题快速响应
- **一站式服务** — 支持 Claude Code、CodeX、Gemini CLI 等多种 AI 编程工具

## 快速开始

只需四步即可开始使用：

1. **注册账号** — 访问官网完成注册
2. **创建 Key** — 获取您的专属 API 密钥
3. **安装 Node.js** — 配置运行环境
4. **部署工具** — 开始使用 Claude Code 等工具

<Callout type="tip">
  推荐使用我们的 [CC-Switch 配置工具](#)，可以一键完成所有配置。
</Callout>
