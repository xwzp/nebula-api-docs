访问我们的官网 <a href="https://ai.nebulatrip.com/" target="_blank">https://ai.nebulatrip.com/</a>，可自行选择账号名注册。

## 注册步骤

1. 打开<a href="https://ai.nebulatrip.com/" target="_blank">官网</a>链接
2. 点击 **<a href="https://ai.nebulatrip.com/register" target="_blank">"注册"</a>** 按钮

<Callout type="tip">
  推荐使用邮箱注册，方便后续找回密码和接收通知。
</Callout>

## 注册后

注册成功后，您可以：

1. **<a href="https://ai.nebulatrip.com/console/token" target="_blank">创建专属 Key</a>** — 获取 API 密钥
2. **<a href="https://ai.nebulatrip.com/console/topup" target="_blank">充值</a>** — 为账户充值
3. 开始使用 AI 编程工具

## 常见问题

### 收不到验证邮件？

- 检查垃圾邮件文件夹
- 确认邮箱地址输入正确
- 等待 1-2 分钟后重试
- 发送邮件获取帮助: support.ai@nebulatrip.com

### 忘记密码怎么办？

在登录页面点击 **<a href="https://ai.nebulatrip.com/reset" target="_blank">"忘记密码"</a>**，通过注册邮箱重置密码。

---

## 安全建议

<Callout type="important">
  - **不要** 将 Key 分享给他人
  - **不要** 将 Key 提交到代码仓库
  - **不要** 在公开场合展示 Key
  - 使用环境变量或配置文件存储 Key
  - 定期更换 Key 提高安全性
</Callout>
